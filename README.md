Since this project relies completely on vanilla web technologies, no build tools or package managers are required.
Clone or download the repository.
Ensure all four core files (index.html, style.css, script.js, favicon.svg) are in the same directory.
Open index.html in any modern web browser.
Pro-Tip: For the best development experience, use an extension like VS Code's "Live Server" to instantly see your changes as you edit the code.

Customization
If you want to tweak the design, the colors are heavily reliant on CSS gradients and box-shadows. You can easily modify the frontend.
